**Typify** is a sleek, minimalist typing application built for performance enthusiasts. It combines modern "Glassmorphism" aesthetics with professional-grade tracking to help users master their typing speed and accuracy in a focused environment.



---

## ‚ú® Features

* **Precision WPM Logic:** Unlike basic trackers, Typify calculates **Net WPM**, ensuring speed is only rewarded when accuracy is maintained.
* **Live Analytics:** Dual real-time line charts (powered by Chart.js) visualize your **WPM** and **Accuracy** trends as you type.
* **Interactive 3D Visuals:** A signature spinning neon cube that reacts to user interaction.
* **Hidden Branding:** Discover the "Typify" logo through a unique "flashlight" reveal effect when your cursor hovers near the cube.
* **Customizable Experience:**
    * **Multiple Cursors:** Choose between Block, Line, Outline, or Underline caret styles.
    * **Variable Timers:** Practice in 15, 30, 60, or 120-second bursts.
* **Post-Test Breakdown:** A comprehensive result screen with a detailed performance graph and character-level accuracy stats.
